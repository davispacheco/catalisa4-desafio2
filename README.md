*<h1>DESCRIÇÃO<h1>*

Faça um sistema responsável por registros de vendas de uma organização. Toda venda possui um cliente, vendedor responsável, valor a ser pago e data de registro. 

*<h2>Entrega Mínima:<h2>*

:small_blue_diamond: O projeto permite cadastrar diversas vendas no sistema. 

:small_blue_diamond: O sistema não permite cadastrar vendas para clientes não cadastrados. 

:small_blue_diamond: O sistema não permite cadastrar vendas de vendedores não cadastrados.

:small_blue_diamond: O sistema permite listar todas as vendas cadastradas. 

:small_blue_diamond: O sistema permite listar todos os vendedores cadastrados. 

:small_blue_diamond: O sistema permite listar todos os clientes cadastrados. 

*<h2>Entrega média: <h2>*
  
:small_blue_diamond: O sistema não permite cadastrar clientes com e-mail inválido (sem @). 
  
:small_blue_diamond: O sistema não permite cadastrar vendedores com e-mail inválido (sem @). 
  
:small_blue_diamond: O sistema não permite cadastrar clientes com CPFs repetidos. 

:small_blue_diamond: O sistema não permite cadastrar vendedores com CPFs repetidos. 
  
:small_blue_diamond: O sistema não permite cadastrar clientes com e-mails repetidos. 
  
:small_blue_diamond: O sistema não permite cadastrar vendedores com e-mails repetidos. 
  
*<h2>Entrega Máxima: <h2>*
  
:small_blue_diamond: O sistema permite pesquisar todas as compras de um cliente em específico através de seu CPF. 
  
:small_blue_diamond: O sistema permite pesquisar todas as vendas de um vendedor em específico através de seu e-mail.
